# Práctica 1: Diagramas de Flujo y Pseudocódigo

Este repositorio contiene la resolución de los algoritmos propuestos en la primera práctica de **Lógica y Algorítmica**, cubriendo estructuras de control, bucles y métodos de ordenación.

## 📝 Enunciados y Requisitos de la Práctica

### 1. Estructuras Selectivas
* **Ejercicio 1: Clasificar Huracanes**: Algoritmo para identificar la categoría (1-5) según la escala Saffir-Simpson basada en la velocidad del viento.
* **Ejercicio 2: Área de un Triángulo**: Cálculo del área mediante la **Fórmula de Herón** utilizando el semiperímetro ($s$).
* **Ejercicio 3: Taxi**: Cálculo de tarifa con base fija (30€) y recargos por exceso de kilometraje tras superar los 30 km y los 100 km.
* **Ejercicio 4: Comparar Temperaturas**: Detección de coincidencias entre tres valores (igualdad total, parcial o diferencia absoluta).

### 2. Estructuras Repetitivas
* **Ejercicio 5: Máximo de Conjunto Conocido**: Búsqueda del valor mayor solicitando previamente el número total de datos a introducir.
* **Ejercicio 6: Máximo de Conjunto No Conocido V1**: Identificación del máximo con finalización de lectura mediante el valor centinela **-999**.
* **Ejercicio 7: Máximo de Conjunto No Conocido V2**: Cálculo de **media aritmética** y **desviación estándar** para datos en el rango [-50, 50], finalizando con **-99**.
* **Ejercicio 8: Tablas de Multiplicar**: Generación sistemática de las tablas de los 10 primeros números naturales.

### 3. Ordenación y Búsqueda
* **Ejercicio 9: Ordenación por Inserción**: Algoritmo que inserta cada elemento en su lugar correcto desplazando los elementos posteriores de la parte ordenada.
* **Ejercicio 10: Ordenación por Selección**: Intercambio del elemento más pequeño encontrado con la posición actual hasta completar la lista.

<h2 id="english">English</h2>

# Practical 1: Flowcharts and Pseudocode

This repository contains the resolution of the algorithms proposed in the first **Logic and Algorithmics** practical session, covering control structures, loops, and sorting methods.

## 📝 Assignment Prompts and Requirements

### 1. Selective Structures
* **Exercise 1: Hurricane Classification**: Algorithm to identify categories (1-5) based on the Saffir-Simpson scale and wind speed.
* **Exercise 2: Triangle Area**: Calculation using **Heron's Formula** and the semi-perimeter ($s$).
* **Exercise 3: Taxi**: Fare calculation with a flat rate (30€) and surcharges for mileage exceeding 30 km and 100 km.
* **Exercise 4: Temperature Comparison**: Detection of matches among three values (total equality, partial match, or absolute difference).

### 2. Repetitive Structures
* **Exercise 5: Maximum of a Known Set**: Searching for the highest value after requesting the total number of data entries.
* **Exercise 6: Maximum of an Unknown Set V1**: Finding the maximum with input ending via the **-999** sentinel value.
* **Exercise 7: Maximum of an Unknown Set V2**: Calculation of **arithmetic mean** and **standard deviation** for data within [-50, 50], ending with **-99**.
* **Exercise 8: Multiplication Tables**: Systematic generation of tables for the first 10 natural numbers.

### 3. Sorting and Search
* **Exercise 9: Insertion Sort**: Algorithm that inserts each element into its correct place by shifting subsequent elements in the sorted part.
* **Exercise 10: Selection Sort**: Swapping the smallest element found with the current position until the list is fully sorted.
